# 思源黑体 (Source Han Sans) CN

从 Adobe 官方仓库下载的简中子集（SubsetOTF/CN）：

| 文件 | 字重 | 大小 |
|---|---|---|
| `SourceHanSansCN-Regular.otf` | 400 Regular | ~8 MB |
| `SourceHanSansCN-Bold.otf` | 700 Bold | ~8 MB |

## 来源

- 官方仓库：https://github.com/adobe-fonts/source-han-sans
- 下载路径：`SubsetOTF/CN/` 分支 `release`

## 许可

**SIL Open Font License 1.1** — 免费商用、可嵌入、可再分发。
完整协议见 https://github.com/adobe-fonts/source-han-sans/blob/master/LICENSE.txt

## 为什么选这个字体

- 免费可商用，不会在打包分发时产生版权风险（苹方/微软雅黑等系统字体不可分发）
- 简中字符全覆盖，适合产品视频字幕
- 文件大小可控（对比 Super OTC ~100MB）
